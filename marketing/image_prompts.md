# Marketing Prompts for AI Image/Asset Generation

## Prompt 1 — Hero Image (архитектурная диаграмма)

Generate a dark-themed technical illustration showing:
- Left side: a ServiceNow interface with catalog items, workflow diagrams, and code panels, rendered in ServiceNow's blue/gray colors
- Right side: multiple AI agents (geometric shapes with glowing nodes) connected by data flow lines
- Center: a bridge/arrow labeled "AI Migration Architect" transforming the left into the right
- Style: enterprise tech illustration, dark navy background, cyan and amber accent colors, clean geometric shapes, no text labels — visual metaphor only
- Aspect ratio: 16:9, suitable for LinkedIn header or presentation cover

## Prompt 3 — Pipeline Visualization

STAGE 1: "DISCOVER" — magnifying glass over ServiceNow logo, data flowing into a database icon
STAGE 2: "ANALYZE" — gears/cogs processing data, scorecards appearing (0-100 health scores)
STAGE 3: "GENERATE" — documents emerging from a printer/AI icon (TOR, specs, roadmap, risk register)

Each stage connected by flowing arrows (data pipeline). Color coding: blue → amber → green.
Dark background. Style: modern SaaS product illustration, similar to Stripe/AWS product pages.
