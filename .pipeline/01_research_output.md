### 🥇 Продукт 1: Catalog Workflow Intelligence (CWI)
**Кодовое название:** `catalog-orchestrator-ai`

#### Проблема
197 catalog items, но только 6 workflows. **97% catalog items работают без оркестрации** — ручные fulfillments, непрозрачные SLA, отсутствие self-healing. Reddit подтверждает: Zero to Hero означает начинать с каталога.

#### Решение
Инструмент сканирования catalog → ML-классификация паттернов (approval, provisioning, notification, escalation) → генерация AI Agent Studio workflows с zero-code конфигурацией.

#### Рынок
- **TAM:** 7,000+ enterprise ServiceNow customers с разросшимся каталогом
- **SAM:** Platform Owners, ITSM Managers, ServiceNow Architects
- **Ценообразование:** $15–25K / инстанс (audit) + $40–80K (migration + agent generation)
- **ROI:** 5–8 месяцев (deflection 40–55% по Australia AI benchmarks)

#### Технический подход
```
sc_cat_item REST scan
         ↓
Pattern classifier (Approval / Provisioning / Notification / Escalation)
         ↓
AI Agent Studio workflow generation
         ↓
4 built-in agent topologies: Approval assistance, Issue Readiness, Request status, Domain Separation
         ↓
Workflow Studio (unified flows/playbooks/actions/decision tables)
```

**Архитектура агентов:**
- **Orchestrator Agent** → принимает запрос из catalog
- **Specialist Agents:** Approval Agent, Provisioning Agent, Notification Agent, Escalation Agent
- **Каждый specialist** = Now Assist skill, конфигурируемый через AI Agent Studio

### 🥈 Продукт 2: Integration Hygiene & Modernization Suite (IHMS)
**Кодовое название:** `integration-zombie-hunter`

#### Проблема
11 legacy REST интеграций, включая мёртвые endpoints (Yahoo Finance, Firebase Cloud Messaging x2 — один из них legacy). Нет централизованного API Catalog, нет health monitoring, security credentials разбросаны.

#### Решение
Автоматический аудит `sys_rest_message` → endpoint reachability testing → security scoring (credential rotation, HTTPS validation) → миграция на Integration Hub + API Catalog.

#### Рынок
- **TAM:** ServiceNow customers с >5 интеграциями (~65% базы)
- **SAM:** Integration Architects, Security Teams, Platform Owners
- **Ценообразование:** $20K (audit + health dashboard) + $30K (remediation + migration)
- **Уникальная ценность:** ServiceNow НЕ предоставляет встроенного health monitoring для REST integrations — это gap.

#### Технический подход
```
sys_rest_message scan (active=true)
         ↓
Endpoint reachability test (httpx HEAD/GET)
         ↓
Security scoring: HTTPS=OK, CredentialAge, EndpointEntropy
         ↓
Integration Hub recipe generation
         ↓
API Catalog registration (autogenerated OpenAPI specs)
         ↓
BYOK LLM via Generative AI Controller для anomaly detection
```

**Встроенная проверка дублирования (Phase 0):**
- ServiceNow имеет `Integration Hub`, `System Web Services`, но НЕ имеет health scoring для REST integrations
- Наличие `Yahoo Finance` в интеграциях — явный индикатор зомби-интеграций
- Перекрытие с built-in < 10% → безопасно

### 🥉 Продукт 3: Script Debt Analyzer → Now Assist Skill Generator (SDA)
**Кодовое название:** `script-debt-2-ai`

#### Проблема
4,775 script includes + 5,654 business rules = **~10,429 серверных скриптов** с высоким техническим долгом. Global scope, hardcoded values, отсутствие документации, legacy GlideRecord паттерны. Reddit обсуждает Now Assist vs AI startups — именно здесь SN может проигрывать agility из-за script debt.

#### Решение
AI-powered AST-анализ server-side кода → обнаружение уязвимостей, dead code, hardcoded values → безопасная генерация Now Assist Skills и Workflow Studio actions.

#### Рынок
- **TAM:** Все enterprise ServiceNow instances с custom development
- **SAM:** ServiceNow Developers, Platform Architects, Security Auditors
- **Ценообразование:** $10K (audit report) + $50K (migration factory)
- **Уникальная ценность:** Единственное решение, мигрирующее script debt в AI-first архитектуру

#### Технический подход
```
sys_script_include + sys_script download
         ↓
AST parsing (Esprima / custom GlideParser)
         ↓
Vulnerability detection (SQL injection, hardcoded sys_ids, global scope abuse)
         ↓
Pattern matching: Can this script become a Now Assist Skill?
         ↓
Skill generation for AI Agent Studio
         ↓
Workflow Studio action templates
         ↓
Refactoring playbook (human-in-the-loop)
```

**Агент-топологии:**
- **Code Analyzer Agent** — AST parsing + scoring
- **Migration Advisor Agent** — генерирует refactoring roadmap
- **Skill Generator Agent** — преобразует script logic в Now Assist skill
- **Validation Agent** — тестирует сгенерированные skills в sandbox PDI
